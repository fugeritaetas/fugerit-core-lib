Fugerit Java Core Library (collections of classes and tools for the java programming language version 1.6+)

This project is a collections of classes and tools. The packages found in the collections may be part of three kind of software :

  • Functions - Collections of functions implementing functionality beyond that of java SDK.
    • IO functions (For stream and File)
  • Utilities - Software that can be used as stand alone.
    • JSQL - Simple RDBMS jdbc connections tool (inspired from psql, postgres default command line tool)
  • Framework - Module that can be used as base for building utilities and applications.
    • Navmap - Control framework inside an MVC J2EE pattern.
    • Doc - Publish frameork to generate documents in various formats.
